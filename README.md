# 🎬 Netflix Clone UI

A simple and responsive **Netflix Homepage Clone** built using **HTML & CSS**.
This project replicates the look and feel of Netflix's landing page with sections like trending movies, comedy, and horror categories.

---

## 🚀 Features

* 🎥 Hero section with featured movie
* 📺 Horizontal scrolling movie rows
* 🔥 Trending, Comedy & Horror sections
* 🎨 Smooth hover effects
* 📱 Responsive layout (basic)
* 🧭 Clean footer section

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Hover Effects)
